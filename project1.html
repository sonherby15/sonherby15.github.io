UCF Together

UCF Together was created to be an all-in-one site for UCF student interaction. It can more easily be thought of as a combination of a chat room, study room, and trading room, for UCF students only. It hosted many other features, like messaging, on-campus mapping features, public-alert systems, and more. It was hosted on Wordpress and supported by a number of homemade plugins created from CSS and Python.

I learned many skills from this project, including:

  • HTML/CSS: The website, while hosted on Wordpress, utilized many homemade plugins crafted with CSS. HTML was also used for minor markup and styling purposes.
  • Automatic Website Monitoring: An important facet of the website was moderation. Coding a Python program to automate this proccess 24/7 was a great learning experience.
  • Collaboration: The project was completed over the course of four months in a team of five. Deligating responsibilities and creating schedules helped the final product.
