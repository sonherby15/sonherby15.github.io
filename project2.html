Degree Audit Program

For one of my classes in Sophomore year, I was tasked with creating a degree audit program, that could atomatically assess what classes were needed to graduate, and generate a rough degree audit including a class schedule and cost analysis.

I learned many Java skills from this project, including:

  • Input/Output Mechanics: Working with the I/O stream in Java was a large undertaking for me, and a great learning experience.
  • SQL: Working with SQL packages was a new challenge, as well as working with databases as a whole, but I learned many tricks on how to work with them.
  • Exception Handling: Managing errors on a project this large took lots of effort, but forced me to learn many of Java's tools to solve them.
